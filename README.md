# AI Natural Language SQL Agent

An AI-powered system that allows users to query a database using natural language.
The system converts human language questions into SQL queries and retrieves data from a Supabase PostgreSQL database using n8n workflow automation.

---

# Project Overview

This project integrates:

* **n8n workflow automation**
* **Google Gemini AI model**
* **PostgreSQL database**
* **Supabase database hosting**

Users can simply ask questions such as:

* Show all customers
* Get freight cost details
* Show forecast monthly data
* List products with highest gross price

The AI automatically converts the question into SQL and executes it.

---

# Architecture

User Question
↓

↓
AI Agent (Google Gemini)
↓

↓
Execute SQL Query Node
↓
Supabase PostgreSQL Database
↓
Return Results

* n8n – workflow automation platform

* PostgreSQL – relational database

# Database Tables

The project queries data from the following tables in Supabase:

* customer
* fact_forecast_monthly
* fact_freight_cost
* fact_gross_price

Example SQL generated by the AI:

SELECT * FROM customer;

SELECT * FROM fact_freight_cost;

SELECT * FROM fact_gross_price ORDER BY gross_price DESC;

# Setup Instructions

1. Clone the repository

git clone https://github.com/yourusername/ai-natural-language-sql-agent.git

2. Install and run n8n

https://docs.n8n.io

3. Import the workflows

Open n8n
Import the workflow JSON files from the workflows folder.

4. Configure API Keys

Add your Google Gemini API key.

5. Connect Database

Configure PostgreSQL connection using your Supabase database credentials.

6. Run the workflow

Send a natural language query to the AI agent.
